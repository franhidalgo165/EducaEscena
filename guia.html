3. Gestión y Automatización

  • Autogestión: Consultas dinámicas que agrupan meses y sedes automáticamente.
  • GitHub (Tu flujo): Todo cambio en `.php` -> `Commit` en VS Code -> `Sync` a la nube.
  • Plugins Clave: Elementor, WooCommerce, Forminator, Yoast SEO y Tribe Events.

🚀 Roadmap de Mejoras Técnicas

  • Optimización: Implementar transitorios (transients) para consultas pesadas de eventos.
  • Emails: Configurar WP Mail SMTP para estabilidad en notificaciones de reserva.
  • Dashboards: Crear panel administrativo personalizado para lectura de registros.
  • Seguridad: Refinar el archivo `.gitignore` para excluir carpetas pesadas/sensibles (ej: `/uploads`, `/node_modules`, `.env`).
